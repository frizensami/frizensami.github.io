# Overview

Singlang (at <a href="https://sriramsami.com/singlang/" target="_blank">sriramsami.com/singlang</a>) is a small programming language built around Singlish syntax.

The implementation is a real interpreter rather than a string-replacement layer over another language. The original implementation is written in Haskell using Alex and Happy for lexing and parsing, and the browser playground mirrors the current language behavior so it can run entirely on GitHub Pages.
